# Microeconomics

## Module 1: Basic Concepts (8-14%)

### Scarcity

- Limited resources ≠ unlimited wants

### Opportunity Cost

- Next best alternative

- \*\*Trap\*\*: Only highest valued

### PPC

- On: efficient

- Inside: inefficient

- Outside: unattainable

- \*\*Shift\*\*: growth

### Comparative Advantage

- Lower OC

- \*\*Trap\*\*: Not absolute

## Module 2: Supply \& Demand (15-20%)

### S\&D Basics

- Demand: P↓ → Qd↑

- Supply: P↑ → Qs↑

- Eq: Qd=Qs

### Elasticity

- Ed = (%ΔQ/%ΔP)

- Elastic >1

- Inelastic <1

- \*\*TR\*\*: elastic P↑TR↓; inelastic P↑TR↑

### Surplus

- CS: under D above P

- PS: above S below P

### DWL

- Taxes/ceilings/floors

### Incidence

- Inelastic bears more

## Module 3: Consumer Choice (5-10%)

### Utility

- Max: MUx/Px = MUy/Py

- Diminishing MU

### Price Effects

- \*\*Substitution\*\*: relative price

- \*\*Income\*\*: purchasing power

## Module 4: Production \& Costs (10-15%)

### Production

- TP/MP/AP

- Diminishing MP

### Costs

- TC=FC+VC

- ATC=TC/Q

- MC=ΔTC/ΔQ

- MC crosses mins

- \*\*Shutdown\*\*: P<AVC

## Module 5: Firm Behavior \& Structure (23-33%)

### Profit Max

- MR=MC (all firms)

### Perfect Competition

- Price taker (horizontal D)

- SR: profit/loss

- \*\*LR\*\*: P=min ATC=MC

### Monopoly

- MR<D

- \*\*DWL\*\*: P>MC, Q<Qc

### Monopolistic Comp

- LR: zero profit + excess capacity

### Oligopoly

- Game theory

- Nash equilibrium

## Module 6: Factor Markets (8-14%)

- MRP = MP×MR (P in comp)

- Hire: MRP = wage

## Module 7: Market Failure (10-16%)

### Externalities

- Neg: MSC>MPC

- Pos: MSB>MPB

### Pigouvian

- Tax (neg), subsidy (pos)

### Public Goods

- Nonrival + nonexcludable

- \*\*Free rider\*\*

### Commons

- Rival + nonexcludable

- \*\*Tragedy\*\*

## Critical Formulas

- 1. Ed=(%ΔQ/%ΔP)

- 2. TR=P×Q

- 3. ATC=TC/Q

- 4. MRP=MP×P (comp)

- 5. Profit=(P-ATC)×Q

## CLEP Traps

- 1. Shifts vs movements

- 2. Ceiling/shortage floor/surplus

- 3. MR=MC all firms

- 4. Monopoly MR<D

- 5. Shutdown P<AVC

- 6. Elasticity % changes

- 7. Utility MU/P not MU
